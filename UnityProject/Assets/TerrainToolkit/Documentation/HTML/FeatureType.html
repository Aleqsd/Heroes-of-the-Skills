TerrainToolkit.FeatureType

Enumeration

Used by VoronoiGenerator to define the type of interpolation used to generate features.

Values

  • FeatureType.Mountains - Interpolated linearly
  • FeatureType.Hills - Interpolated along a sine curve
  • FeatureType.Plateaus - Interpolated along a tangent curve
